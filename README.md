# 🌟 Berhu - Plataforma de Transformação Espiritual

## 📋 Sobre o Projeto

Berhu é uma plataforma completa de transformação espiritual e terapêutica, criada por Fabiana Berkana. O sistema oferece uma experiência imersiva para acompanhamento da evolução pessoal, acesso a conteúdos exclusivos e agendamento de sessões terapêuticas.

### 🏠 Área de Membros

- **Perfil do Usuário**: Gerenciamento de dados e histórico

- **Conteúdo Exclusivo**: Cursos, meditações e tratamentos
